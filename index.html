what is vesper

Vesper is a lightweight configuration and event bus library for Java applications. Zero dependencies, minimal boilerplate, and a clean API.

features

  • YAML / JSON configuration parsing with strong typing
  • Simple, annotation‑driven event system
  • Built‑in resource reloading and hotswap
  • Designed for Minecraft mods, Discord bots, and standalone projects

get started

Add the Maven repository and the dependency to your build file.

Repository URL: https://maven.codexbat.dev/releases

Maven

<repositories>
    <repository>
        <id>codexbat-maven</id>
        <url>https://maven.codexbat.dev/releases</url>
    </repository>
</repositories>

<dependency>
    <groupId>dev.codexbat</groupId>
    <artifactId>vesper-core</artifactId>
    <version>1.0.0</version>
</dependency>

Gradle (Kotlin DSL)

repositories {
    maven {
        url = uri("https://maven.codexbat.dev/releases")
    }
}

dependencies {
    implementation("dev.codexbat:vesper-core:1.0.0")
}
